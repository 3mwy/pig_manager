# pig manager
## 前言
> 本项目致力于开发一套完整的猪场数据管理系统，采用现阶段流行技术，以下`pm`作为项目简称
---
## 项目介绍
> `pm`项目为一套猪场数据管理系统，包括前台数据管理系统以及后台账号管理系统，基于Spring Boot+MyBatis实现。前台包括首页用户登录、猪只搜索、猪只管理模块。后台包括用户管理、权限管理等模块。
